## name the color in image, map every RGB value to a 11 specified  color types

type -- color
 0   -> black  
 1   -> blue  
 2   -> brown  
 3   -> grey  
 4   -> green  
 5   -> orange  
 6   -> pink  
 7   -> purple  
 8   -> red  
 9   -> white  
 10  -> yellow
